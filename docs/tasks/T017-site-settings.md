# Task T017: Global Site Settings

## Title

Global Website Settings

## Goal

Provide a central CMS page for managing global website configurations such as branding, SEO defaults, and contact information.

## Requirements

- **CMS Settings Page**:
  - **General**: Website Name, Logo Upload, Favicon Upload.
  - **SEO Defaults**: Default SEO Title, Default SEO Description (used when specific page SEO is missing).
  - **Contact**: Contact Email, Social Media Links (Facebook, Twitter, etc.).
- **Frontend Integration**:
  - Update the site logo in the header and footer.
  - Update the site title in the browser tab.
  - Populate social media icons in the footer with the configured links.

## Verification Steps

1. Navigate to "Settings" in the CMS.
2. Upload a new website logo and save.
3. Update the "Default SEO Description" and "Website Name".
4. Add a link to the website's Facebook page.
5. Verify the new logo appears on the frontend header.
6. Check the browser tab title to ensure it includes the new Website Name.
7. Verify the Facebook icon in the footer links to the correct URL.
